# 设计文档 - 孙颖莎人物志网站

## 设计理念

本网站以"运动之美"和"冠军风采"为核心设计理念，通过现代化的视觉语言和流畅的交互体验，展现孙颖莎作为世界顶级乒乓球运动员的卓越成就和个人魅力。

## 视觉设计规范

### 色彩系统

#### 主色调
- **中国红** `#DC143C` - 代表激情、力量和荣誉
- **冠军金** `#FFD700` - 代表成就和辉煌
- **深空蓝** `#0A1929` - 作为背景色，营造专业感

#### 辅助色
- **霓虹蓝** `#00D9FF` - 用于特效和强调
- **荧光绿** `#39FF14` - 用于动态元素
- **渐变紫** `#8B5CF6` → `#EC4899` - 用于背景渐变

#### 中性色
- **纯白** `#FFFFFF` - 文字和背景
- **深灰** `#1F2937` - 次要文字
- **浅灰** `#F3F4F6` - 背景区域

### 字体系统

#### 中文字体
- **标题字体**：思源黑体 Bold / 阿里巴巴普惠体 Bold
- **正文字体**：思源黑体 Regular / 阿里巴巴普惠体 Regular
- **数字字体**：DIN Alternate（用于数据展示）

#### 英文字体
- **标题**：Poppins / Montserrat
- **正文**：Inter / Roboto

#### 字号规范
- **超大标题**：64px - 80px（Hero区域）
- **大标题**：48px - 64px（页面标题）
- **中标题**：32px - 40px（区块标题）
- **小标题**：24px - 28px（卡片标题）
- **正文**：16px - 18px
- **辅助文字**：14px

### 间距系统

基于 8px 网格系统：
- **XS**: 4px
- **S**: 8px
- **M**: 16px
- **L**: 24px
- **XL**: 32px
- **XXL**: 48px
- **XXXL**: 64px

## 动画设计规范

### 页面过渡动画

#### 进入动画
- **淡入**：`fadeIn` - 0.6s ease-out
- **滑入**：`slideIn` - 0.8s ease-out
- **缩放**：`scaleIn` - 0.5s ease-out
- **旋转**：`rotateIn` - 0.6s ease-out

#### 退出动画
- **淡出**：`fadeOut` - 0.4s ease-in
- **滑出**：`slideOut` - 0.5s ease-in

### 微交互动画

#### 按钮动画
- **悬停**：背景色渐变 + 轻微放大（scale 1.05）
- **点击**：缩小反馈（scale 0.95）
- **加载**：旋转动画

#### 卡片动画
- **悬停**：上浮效果（translateY -10px）+ 阴影加深
- **点击**：轻微缩放反馈

#### 滚动动画
- **触发条件**：元素进入视口
- **动画类型**：淡入 + 从下方滑入
- **延迟**：根据元素位置错开延迟时间

### 特效动画

#### 粒子系统
- **类型**：乒乓球轨迹、星星闪烁、光点流动
- **性能**：使用 Canvas 或 WebGL 优化

#### 背景动画
- **渐变流动**：CSS渐变 + 动画
- **波浪效果**：SVG路径动画
- **视频背景**：循环播放，支持静音

## 响应式设计

### 断点设置
- **移动端**：< 768px
- **平板**：768px - 1024px
- **桌面**：> 1024px
- **大屏**：> 1440px

### 适配策略

#### 移动端
- 单列布局
- 简化动画效果
- 触摸友好的交互
- 优化图片大小

#### 平板
- 两列布局
- 保留核心动画
- 适中的字体大小

#### 桌面
- 多列布局
- 完整动画效果
- 丰富的交互细节

## 性能优化策略

### 图片优化
- 使用 WebP 格式
- 响应式图片（srcset）
- 懒加载（Intersection Observer）
- 图片压缩优化

### 动画优化
- 使用 `transform` 和 `opacity`（GPU加速）
- 避免触发重排和重绘
- 使用 `will-change` 提示浏览器
- 限制同时运行的动画数量

### 代码优化
- 代码分割（Code Splitting）
- 按需加载组件
- 使用 React.memo 优化渲染
- 虚拟滚动（长列表）
